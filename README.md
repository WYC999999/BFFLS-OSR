## “爬树校长”刘忠毅：把体育课种进自然里
午间走进丰台外国语学校，正看到刘忠毅校长站在云梯下组织同学们攀爬云梯，脚上的足球鞋沾满灰尘，脸上洋溢着笑容，和学生们打成一片。因为该校在全国率先开设“爬树课”，刘校长被媒体朋友风趣的称为“爬树校长”，这位“爬树校长”可不简单，多年的采访经历告诉我，这是一位务实创新、科学施教、勇于实践的陶行知式的好校长。

对于我们这个年纪的人来说，爬树很简单，上树掏鸟窝、捅马蜂窝、摘果子，这是最基本的技能，不仅男孩子可以，女孩子也是赛猴子。但对于当下的儿童来讲，这确实是个困难，没有几个孩子会爬树，上肢力量匮乏，四肢不协调，很难爬到树上去。其实不仅爬树，跳高、跳远、单杠、鞍马等众多活动都已经成为孩子们无法逾越的困难。教育部早就关注到了学生体质健康的问题，也一直在努力破题，这其实是当前社会面临的一个难题。

刘忠毅校长对于开设爬树课已经酝酿很久，也采取了步步为营的措施。他首先是把学生们从教室解放出来，吸引到操场上。因此在4年前，他率先在全国开始了课间时长改革的探索，他力排众议，把每节课间都从10分钟延长到15分钟， 学生每节课间都可以到户外运动，奔跑、跳跃、追逐，几年下来，眼睛明亮了、身体强壮了、心理问题都缓解了。这短短的五分钟变化，其实是教育理念和育人观念的变革，是真正以儿童为中心的教育理念的践行。

有了课外活动时间还不够，还要教会孩子们玩，这才是让课间灵动起来的核心。每天中午正副班主任要教学生一项运动技能，两周换一样，每学期可以学会5、6项运动，每个孩子都能找到自己的热爱：单杠倒挂、双杠支撑、云梯猴子爬、鞍马腾跃…… ，“孩子不是不爱运动，是没找到自己喜欢的‘打开方式’。”刘忠毅说。如今下课铃一响，运动器材区就成了“欢乐场”，有学生能在云梯上“横向飞移”，也有孩子抱着双杠练“支撑倒立”。

学生爬云梯的方式多样，有的学生自创“螃蟹爬”，双脚支撑两边立棍，交替上升可以爬上去；有的学生自创“猴子爬”，双手抓住铁棍，双脚夹住铁棍往上爬，发现爬不上去，把裤子卷到膝盖处，用腿部肌肉的摩擦力帮助爬升。在这些活动中，学生的智力和能力都得到提升，这是自身体验的获得，是任何课堂都无法教会学生的。正是这些运动，学生们的四肢力量得到了提升，协调性得到了锻炼，才让刘校长有了到公园里开爬树课的念头。

2024年暑假，刘忠毅带着老师们在校园周边踩点时，河畔公园的一片矮树林让他停住了脚：“树不高、枝杈稳，这不就是天然的体育器材？”彼时，他正为“如何让体育走进生活，成为孩子们随时可以参与的运动”犯愁，这片树林成了破题的钥匙。“把体育课搬进公园”的提案，最初在教职工会上引发过争议：“安全怎么保障？教学目标怎么定？”刘忠毅的回应很直接：“先试一节，让孩子说了算。”9月的第一堂爬树课上，原本拘谨的学生们抱着树干笑作一团，有孩子攥着粗糙的树皮喊“比单杠好玩一百倍”——这让刘忠毅笃定：“自然里的运动，才是孩子愿意‘扑进去’的课堂。”当然，开设课程也是艰难的，刘校长是学校里第一个爬树的“学生”。以前刘校长经常爬树，但随着环境和年龄的增长，已经几十年没爬过树了，特别是要想教孩子爬树，就要知道爬树会出现哪些问题，刘校长就成了学校里的第一个爬树学生。刘校长双手抱住树干、双腿夹着树干往上攀爬，怕了1米多就气喘吁吁了，特别是下来后，他发现双腿都皮肤擦破了。于是，刘校长意识到，要想爬好树，安全是第一位的。为了这堂“野课”，刘忠毅带着团队磨出了一套“安全手册”：爬树前必须戴头盔、穿长袖，老师要提前“手测树干结实度”“踩验树杈支撑力”；校内先练云梯“悬挂抓握”“横向位移”，用单杠、双杠打牢上肢力量基础。“不是让孩子瞎爬，是让他们在自然里学技能、练体能。”

丰台外国语学校的学生能爬树，丰外被称为“爬树学校”。现在，每个年级都有几十个孩子能安全、迅速地爬上3米高的大树，孩子们课间在校园内吊单杠、撑双杠、爬云梯，为爬树做准备。每到爬树课，都捋胳膊卷袖子，跃跃欲试。刘忠毅校长说，如今爬树课成了“抢手课”，常有学生追着老师问“下周能爬更高的树吗？”

在刘忠毅看来，爬树课的意义，不止于“学会爬树”。“当孩子抱着树干感受树皮的粗糙，看着叶影在湖面晃成一团，他摸到的是自然，释放的是天性。”这种“校园+公园”的模式，让教育跳出了围墙——学校与公园共建“乐学基地”，除了爬树课，还开发了“自然定向越野”“湖畔平衡木”等课程，把运动、自然、生存技能揉进了日常。刘忠毅说：“教育部要求‘每天两小时体育’，不是让孩子在操场耗够时间，是让他们在喜欢的运动里‘活’起来。”刘忠毅指着公园里追着落叶跑的学生笑，“你看，这才是童年该有的样子——爬过树，踩过叶，把自然的气息揣进兜里，把健康的底子打在年少。要感谢丰台区委王少峰书记，正是因为他推动的自然体育课、乐学公园项目，才让孩子们能走出校园、走进自然，教育理念的变革真的很重要！”

如今，“爬树校长”的名号越传越远，常有其他学校的老师来取经。刘忠毅总说：“改革不用多复杂，只要站在孩子的角度想——他们喜欢跑、喜欢爬、喜欢在自然里撒野，我们就把课堂搬到他们喜欢的地方去。”而那片河畔公园里，又一批孩子正抱着树干，往树杈上攀，阳光穿过叶隙，落在他们笑红的脸上，像极了教育最生动的模样。
