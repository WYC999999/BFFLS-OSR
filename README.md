# 鸡丁怪谈
> [!CAUTION]
> 谁也不知道不遵守以下规则会发生什么
### 1.当你进入鲁味小馆，大鸭梨，北平食府等具有宫保鸡丁菜品的餐厅，你必须食用宫保鸡丁，否则平浩恩会在0.001秒内定位并杀了你
### 2.当你进入上述餐厅，你必须按照以下要求下单宫保鸡丁:有花生，有花椒，咸辣口味，鸡肉很嫩。如果厨师没有按照这些要求炒菜，请你立刻到后厨与厨师吵架，不用担心，当厨师拒不听从你的意见时，平浩恩会立刻到达你附近并打死厨师
### 3.如果厨师做出了上述菜品，请你务必食用完，否则平浩恩会来打死你
### 4.当你吃完后，请你在餐桌前大喊一声“趴会”，并在桌子上趴下假装睡觉，否则平浩恩会立刻枪毙你并扣除你的社会信用点
### 5.完成以下动作后，当你听到其他人谈论时，你必须立刻靠近并大声说“就这个啊”，“我知道这个”，不管你知道不知道你必须这么说，否则后果自负
### 6.当你离开这里时，你必须打着响指并摇摇晃晃的出门，否则平浩恩会在你耳朵旁边打一辈子响指
### 7.扣除社会信用点后，你必须1天3顿食用宫保鸡丁，每当你完成了今天的3顿宫保鸡丁，你的社会信用点会增加5点
### 8.千万不要试图在任何地方诋毁暗区突围或艾尔登法环，平浩恩会立刻飞到你的面前虐杀你
### 9.在餐厅食用宫保鸡丁后，你必须使用科恩币，势力币，CP点，赛季CP点，GP点支付，如果你擅自使用其他货币会被平浩恩当场没收并按照对应游戏内汇率兑换成游戏内货币
### 10.如果你被平浩恩杀死后，你必须在死后的世界里找到9盘宫保鸡丁，将其摆成正方形后坐在正中心打9个响指，死神平浩恩便会降临并复活你，注意，该机会仅有两次，用完后再次死亡便会被做成宫保鸡丁被平浩恩吃掉
